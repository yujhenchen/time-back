## Why

The README currently lists the tech stack in a table but has no visual badges. Badges provide instant recognition of the project's technology choices at a glance, improve credibility, and make the README feel more polished and maintainer-friendly.

## What Changes

- Add a row of tech stack badges to the README header area using [shields.io](https://shields.io) badges
- Each badge represents one technology from the stack (Plasmo, React, TypeScript, Tailwind CSS, shadcn/ui, Zod, etc.)
- Badges link to the respective technology websites or docs
