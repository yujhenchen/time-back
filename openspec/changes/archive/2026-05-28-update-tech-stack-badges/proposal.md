## Why

The current tech stack badges are plain text labels with no logos. Shields.io supports rich icons from Simple Icons for most technologies, making badges more recognizable and visually polished at a glance.

## What Changes

- Replace each plain text tech stack badge with one that includes the official Simple Icons logo
- Use shields.io `logo` parameter for React, TypeScript, Tailwind CSS, shadcn/ui, and Zod
- Plasmo (not in Simple Icons) keeps its plain style but gets a branded purple color
- All badges remain `flat-square` style with branded colors and correct links

## Capabilities

### New Capabilities

- `tech-badge-logos`: Add Simple Icons logo images to each tech stack badge

### Modified Capabilities

- _(none)_

- **Modified file**: `README.md` — update badge URLs in the header
- **No code, dependency, or API changes**
